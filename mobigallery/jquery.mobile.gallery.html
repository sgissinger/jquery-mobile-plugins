HTML5 Attributes

data-uselang Language used by the component
default: 'en'
data-theme Theme swatch applied to fullscreen page buttons
default: 'c'
data-transition The transition used to open the fullscreen page
default: 'none'
data-hashname String appended to the browser's address in the address bar
default: 'gallery'
data-slideenabled When set to false, swipe effects navigation in the fullscreen page are disabled for all browsers, this allow to not preload next and previous images.
Browsers not supporting 3D translations fallback to swipeless effects navigation. Touch moves are always supported whatever this value is true or false
default: true
data-slidefunction CSS3 transition-timing-function property value applied to fullscreen page swipe effect navigation
default: 'ease-out'
data-slideduration CSS3 transition-duration property value applied to fullscreen page swipe effect navigation
default: '.5s'

Localization

Additional languages can be added like the sample below which adds french localization (which is already included at the end of the plugin file)
$.extend($.mobile.gallery.prototype.options.lang, {
    'fr': {
        close: 'Fermer la galerie',
        previous: 'Image précédente',
        next: 'Image suivante',
        on: ' sur '
    }
});
