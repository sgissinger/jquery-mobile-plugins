HTML5 Attributes

data-uselang Language used by the component
default: 'en'
data-theme Theme swatch applied to the file input container
default: 'a'
data-buttontheme Theme swatch applied to the remove button
default: 'b'
data-ismultiple For a given file-set, when this option is set to
  • false, a single file input is managed
  • true, multiple file inputs is managed and when a file is selected in the last file input, a new one is cloned and displayed below it.
default: false
data-afterdomupdate Name of a javascript function called
  • after an existing file input is removed
  • after a new file input is added and displayed
default: 'undefined'

Enable/Disable

File inputs can be enabled/disabled by calling 
.trigger('enable')
or 
.trigger('disable')
methods.
This also allows mobile browsers lacking file upload support to disable the enhanced component when they disable the file input component.
